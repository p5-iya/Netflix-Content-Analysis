# Netflix-Content-Analysis
Project Overview

The Netflix Content Analysis Dashboard is an interactive data visualization project built using Power BI to analyze Netflix's content library. The dashboard provides insights into content distribution, genres, ratings, release trends, and country-wise availability, helping users understand patterns and trends in Netflix's catalog.

Objectives-
Analyze the distribution of Movies and TV Shows on Netflix.
Identify popular genres and content ratings.
Explore content growth trends over the years.
Examine country-wise content availability.
Provide interactive filtering for deeper analysis.

Key Features-
KPI Cards displaying Total Titles, Movies, TV Shows, and Ratings.
Content Distribution Analysis (Movies vs TV Shows).
Genre-wise Content Breakdown.
Country-wise Content Analysis.
Release Year Trends and Content Growth.
Interactive Slicers for filtering by Type, Country, Rating, and Year.

Tools & Technologies-
Power BI
Power Query
Microsoft Excel/CSV Dataset

Insights Generated-
Distribution of Movies and TV Shows across the platform.
Most popular content genres and ratings.
Trends in content releases over time.
Geographic distribution of Netflix content.
Growth of Netflix's content library over the years.

Dataset-
The dataset used for this project contains information about Netflix titles, including:

Title
Type (Movie/TV Show)
Genre
Country
Rating
Release Year
Date Added
Duration

Outcome
This dashboard enables users to explore Netflix content interactively and gain valuable insights through data visualization and analytics. It demonstrates practical skills in data cleaning, transformation, modeling, and dashboard design using Power BI.
